已完成的API
用户中心
  • 注册: POST api/v1/signup
  • 登录: POST api/v1/login
  • 重置密码: PUT api/v1/users/:userId/resetPassword
  • 获取用户信息: GET api/v1/users/:userId
  • 修改用户信息: PUT api/v1/users/:userId
  • 授权Github: GET api/v1/github/auth/:code
  • 获取令牌: GET api/v1/github/token/:githubId
文章
  • 查询文章列表: POST api/v1/posts?page&perPage
  • 新增文章: POST api/v1/post
  • 修改文章: PUT api/v1/posts/:postId
  • 删除文章: DELETE api/v1/posts/:postId
管理中心
  • 查询用户列表: POST api/v1/users?page&perPage
  • 修改用户状态: PUT api/v1/users/:userId/:status
  • 注销用户: DELETE api/v1/users/:userId
